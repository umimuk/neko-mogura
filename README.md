# ねこモグラたたき (Neko-Mogura)

30秒間の制限時間内に、穴から出てくる猫をたたく「モグラたたき」ゲームの猫画像版です。

## 特徴

- **マルチメディア体験**: レインボー猫の登場、進化した「ニャー」という鳴き声（サウンドエフェクト）、振動フィードバックを搭載。
- **予測不能なゲーム性**: 固定間隔ではなく、ランダムなタイミングで猫が出現します。
- **難易度設定**: 「やさしい」「ふつう」「むずい」の3段階から選択可能。
- **オンラインランキング**: GAS (Google Apps Script) を使用した世界ランキング機能を搭載。
- **レスポンシブデザイン**: ダークモードやモバイル（バイブレーション）に対応したプレミアムなUI。

## 遊び方

1. 名前を入力して「スタート」をクリック。
2. 出てくる猫をタップ、またはクリックしてスコアを稼ぎます。
3. **レインボー猫**: 5点獲得の大チャンス！出現時間が短いので、素早くたたきましょう。
4. **コンボ/フィーバー**: 10コンボごとにフィーバーモードに突入し、獲得スコアが2倍になります。
5. 30秒経過後、スコアをランキングに送信できます。

## 開発・統合履歴

- **Subagent 1**: SE・画像・ロジックの各機能の初期実装と統合の試行。
- **Subagent 3**: レインボー猫アセットの生成、サウンドのブラッシュアップ、ロジックの最終調整、および全機能の完全統合。
